# notebooks

本目录存放 Jupyter Notebook 代码，用于快速验证、学习实验和复现教程。

## 目录说明

| 目录 | 内容 |
|---|---|
| `python/` | Python 基础库：python 语法、numpy、pandas、scipy |
| `data/` | 数据源接入与导出：akshare 等 |
| `quant/` | 量化工具库：backtrader、btframe、pyfolio、empyrical |
| `models/` | 机器学习模型训练与验证（空，待补充） |
| `plots/` | 可视化：seaborn 等 |
| `assets/` | 金融产品分析：期权等 |

## 使用方式

需要快速验证某个想法时，在对应目录下新建 notebook 即可。跑通了可以整理成正式笔记，跑不通的删掉也不心疼。
